Keys & access

Three secrets, two sync channels — kids cannot edit the chore catalog without the parent secret.

Secrets

SecretWho has itPowers
Family code All devices Public channel: see published catalog (with prices), mark done, proposals, completions
Parent secret Parents only Admin channel: edit catalog, payday, settings, publish catalog + permissions
Device link One kid device Sign that kid’s completions (HMAC); parent can verify

Two encrypted channels

  • Public room — key from family code only. Kids connect here.
  • Admin room — key from family code + parent secret. Parent devices only.

Even with a hacked kid app, without the parent secret the device never receives admin ciphertext.

Code strength

Codes are 16 random bytes (URL-safe ~22 chars), not short word lists. Brute-forcing stolen relay blobs is not practical.

Catalog publish

Parents edit chores on the admin doc, then publish active chores to the public catalog map. Kids read only the snapshot.

See Kid permissions for what kids are allowed to see in the UI.

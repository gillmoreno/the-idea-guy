Technical documentation

Plain-language docs for everything in this repo — what each piece is, why it was built that way, and where the code lives. Kept in sync with the repo via a manifest, git hooks, and the html-docs Claude Code skill.

The three products

Rooms apps/rooms One installable PWA, many room templates (ChoreBoard, Trip Split, Book Club, Fit Crew, Backlog). Local-first: Yjs CRDTs on each device, E2E-encrypted sync through a dumb relay that never reads your data. Not deployed yet.
Second Brain secondbrain/ Obsidian-style notes vault in the browser — internal links, backlinks graph, full-text search, HTML pages/blocks, AI chat (OpenAI or local Ollama). Same local-first kit. Not deployed yet.
Website frontend/ + backend/ Personal brand site: shipped builds, public idea log, tools & workflow. Next.js static export + Go API behind nginx. Not deployed yet.
